The Mashable News Plugin simply adds a customizable widget which displays the latest Mashable posts. It can be integrated anywhere on your blog. This newsticker shows up the last five or more Mashable News posts. This is a very nice solution for all those interested in keeping tabs on what's going on in the Social Media world, and also to keep tabs on up to the minute releases and upgrades, breaking web news and such.

== Description ==

The Mashable News Plugin simply adds a customizable widget which displays the latest Mashable posts. It can be integrated anywhere on your blog. This newsticker shows up the last five or more Mashable News posts. This is a very nice solution for all those interested in keeping tabs on what's going on in the Social Media world, and also to keep tabs on up to the minute releases and upgrades, breaking web news and such.

If you have tech related questions/support etc, though I don't see why you would (check the read me file included for help), it's really a basic plugin, shoot me a msg on my GuyRo blog @ guyro.com in the comments and i'll do my best to get back to you.

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==

*Could not be simpler*
1. Extract `mashable-rss-plugin.zip`.
2. Upload the `mashable-rss-plugin`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
